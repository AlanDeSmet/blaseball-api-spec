## Field Descriptions

**`id`**: The ID of the standings object.

**`losses`**: An array of team IDs and how many losses each team has.

**`wins`**: An array of team IDs and how many wins each team has.

The following fields will be empty for seasons prior to season 12:

**`runs`**: An array of team IDs and how many total runs each team has collected.

**`gamesPlayed`**: An array of team IDs and how many games each team has completed.
